How It Motivates & Trains

🎯

Gaze-Away Auto-Pause

Pauses playback instantly when the child looks away and resumes once they return, creating an immediate positive reinforcement loop that rewards sustained focus.

🧠

AI-Powered Video Puzzles

Gemini AI scrapes video transcripts on-the-fly to insert custom, context-aware interactive puzzles, turning passive viewing into active learning.

👁️

Double-Gate calibration

Progressive circular Dwell Gates (Question & Answer) prevent accidental clicks ("Midas Touch") while keeping kids relaxed with child-friendly audio read-alouds.

⚙️

Caregiver Remote Control

Whitelist channels, configure learning topics, and set daily watch limits remotely via the caregiver portal, syncing instantly to the device.

Key Core Pillars

GazeAAC combines robust low-level eye-tracking technology with advanced user interface design tailored for ultimate accessibility.

🔌

Tobii Gaze Bridge

Features a custom python-based background service that bridges Tobii Eye Tracker APIs to Electron via WebSockets, supporting local, low-latency gaze polling.

🎯

CP-Optimized Tolerance

Features custom dampening buffers and manual gate switches designed specifically for kids with spasticity, preventing accidental double-clicks and mis-selections.

📁

OBF Grid Compatibility

Fully supports Open Board Format (OBF), allowing custom symbols, vocabulary trees, and grid dimensions to be imported and synced seamlessly.

🤖

Contextual AI Responses

Uses offline-first cache syncing powered by Firebase combined with intelligent AI query engines for quick, contextually relevant conversational responses.

📷

Intelligent Video Augmentation

Features live background camera indexing via FaceNet and MobileNet models, feeding identified objects and face emotions directly into the predictive AI.

👤

Biometrics Caregiver Sync

Allows caregivers to remotely add family face profiles and custom object embeddings inside their caregiver panel, syncing instantly across devices.

System Requirements

  • OS: Windows 10 or Windows 11 (64-bit)
  • Memory: 4 GB RAM minimum (8 GB recommended)
  • Hardware: Tobii Eye Tracker 4C / 5 / PCEye (Optional, supports standard Mouse Emulation mode out-of-the-box!)
  • Frameworks: Microsoft VC++ Redistributable (included in installer)

How to Get Started

  1. Download the installer package (`GazeAAC-Setup-0.2.9.zip`), extract the `.exe` inside, and double-click to install.
  2. Connect your Tobii Eye Tracker (if available) and run the Tobii Calibration utility.
  3. Launch **GazeAAC** from the desktop shortcut. The Python Eye Gaze bridge will start automatically in the background.
  4. If using a standard mouse, the app functions in **Dwell Click** mode—perfect for testing and screen-based navigation!
